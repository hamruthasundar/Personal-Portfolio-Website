🚀 Personal Portfolio Website (Flask)

A responsive and modern personal portfolio website built using **Flask (Python)** with HTML, CSS, and JavaScript. This project showcases my skills, projects, and contact information in a clean and interactive UI.
🧠 Tech Stack

- 🐍 Python (Flask)
- 🌐 HTML5
- 🎨 CSS3
- ⚡ JavaScript (Vanilla JS)
- 🗂 Jinja2 Templates
- 📁 Flask Static File Handling

📁 Project Structure
portfolio/
├── app.py
├── requirements.txt
├── templates/
│ └── index.html
├── static/
│ ├── css/
│ │ └── style.css
│ └── js/
│ └── main.js

✨ Features

- Responsive modern UI
- About section with personal details
- Projects showcase section
- Skills overview
- Contact form
- Flask backend integration
- Clean folder structure
